How do you create a **list** in Python?

Use square brackets:
```python
my_list = [1, 2, 3, 4, 5]
```

===

There are several ways:
- Using square brackets: `[1, 2, 3]`
- Using `list()` constructor: `list((1, 2, 3))`
- Using list comprehension: `[x for x in range(5)]`

Solve for $x$: $2x + 5 = 13$

***

Step by step:
1. Subtract 5 from both sides: $2x = 8$
2. Divide by 2: $x = 4$

**Answer:** $x = 4$

What are the three states of matter?

===

1. **Solid** - Fixed shape and volume
2. **Liquid** - Fixed volume, variable shape
3. **Gas** - Variable shape and volume

> Some scientists also consider **plasma** as a fourth state.
